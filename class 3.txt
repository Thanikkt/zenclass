Linux Commands - 
Structure of a command
Getting help for a command 
man page

one feature of bash - using up arrow to get to the previously used commands
another feature --> use tab to auto complete, use tab tab (tab twice) to list the options with same name
use the last command based on word - ctrl + r --> recursive/ reverse search from history
-r --> recursive, -f --> force, -v --> verbose (debug), -i --> interactive
Information on list command - ls -l  - ll
touch - creating a sample file 
mkdir - create an empty directory
create files like file1, file2, file3 till file5
wild cards - {1..5} --> to create a range from 1 till 5
 .. --> to go one step behind or the parent folder of the current folder 
 . --> to refer to the current working directory
rm --> to delete a file
* --> everything /all
cat --> to print the contents of a file
echo --> to print some string/variable
 >   --> is to forward or write to a file
 >> --> is to append to a file
 which --> to know the binary file of the command
 cp file(src) file1(dest) --> to copy a file 
mv --> move or rename a file or a folder
 in linux the executable --> .bin (binary files)
 
Vi/vim editor / nano editor -->
to edit and modify files
to delete the contents or purge the file without deleting the file--> echo "" > filename
:linenumber --> to go to the line 
dd --> to delete a specific line
Shift + g --> to navigate to end of the file
gg --> to navigate to top of the file
:q! --> to forcefully quit
/string --> to search a string top to bottom
?string --> to search a string bottom to top (reverse)
:%s/to/from --> to search and replace (use /g in the last to completely replace all occurences)
we can use # as a separator instead of /

 
Process management - ps, top, pid, kill, pidoff, pstree, swapon, free
ps - to check all the running/idle/sleeping/zombie process
ps -ef - short description of all running process
ps -aux --> more info on processes
top -> real time process tree view - task manager
pid- process id
ppid - parent process id
