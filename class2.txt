Linux - GRUB loaders

File system structure - 
windows - OS drive name is C:
folders
Linux - OS drive name is / (root)
directories

in windows --> to open a program - .exe
in linux --> bin files (binary)


/etc --> system and software package related config files ==== windows === Program files
/bin --> binary files location == Program files in windows
/opt /var --> optional and variable related ==> windows C:\windows
/boot --> boot files / grub loader files ==> C:\Windows\Boot
/tmp - temperory dir --> never store important configs here
/home --> User related home directory ==> C:\Users
/usr --> user related configs or binaries or variables
/root --> Root(superadmin) user home directory
