Networking - 
we have a structure established to be followed while creating a network which give better explanation of how things are working inside a network --> OSI Open Systems Interconnection Model

in network, everything is sent as packets

tcp - used for wired connections, direct

udp - mostly used for wireless connections / indirect connections

client - making the request , eg - customer in hotel
request - ordering an idli
Server - respoding to the request, hotel

reserved ip address -
